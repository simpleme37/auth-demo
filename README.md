# Auth Demo

一個基礎的用戶認證應用

## Features

- 用戶註冊
- 用戶登入
- 用戶登出
- 使用 bcrypt 進行密碼加密
- 使用 JWT token 進行身分驗證

## Technologies

- 後端：Node.js, Express
- 前端：JavaScript, HTML, CSS
- 密碼加密：bcrypt
- 身分驗證：JWT token
